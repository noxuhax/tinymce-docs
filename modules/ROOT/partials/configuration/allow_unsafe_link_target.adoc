[[allow_unsafe_link_target]]
== `allow_unsafe_link_target`

By default all links with a `target` of `{blanktarget}` will get a `rel` attribute of `noopener`. This will disable access to the `window.opener` object from a child tab/window that will open on click. If this is not something you care about, you can disable this option, by setting it to _false_. Although *we do not recommend* you to do so unless you have other ways of securing links to external pages using target set to `{blanktarget}`.

*Type:* `Boolean`

*Default Value:* `false`

*Possible Values:* `true`, `false`

[discrete]
=== Example: Using `allow_unsafe_link_target`

[source, js]
----
tinymce.init({
  selector: 'textarea',
  allow_unsafe_link_target: true
});
----
